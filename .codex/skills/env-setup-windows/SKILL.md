- 新人机器环境搭建 / 首次打开 Unity / VS 构建失败 / 缺少 .NET Framework Targeting Pack

- VS：安装 “.NET 桌面开发” + `.NET Framework 4.6.2 Targeting Pack`
- SQL Server：Express/Developer/LocalDB
- Python：`3.10+`（用于 `Src/Data/excel2json.py`）
